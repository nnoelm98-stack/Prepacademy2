Systems of Linear Equations
Master two methods for solving systems of two equations — substitution and elimination — and learn to identify when a system has one solution, no solution, or infinitely many solutions.
14 min read3 practice questions+20 pts on completion
Two methods — choose the faster one
A system of two linear equations has two unknowns. To solve it, you must find the values of both variables that satisfy both equations simultaneously. The SAT rewards you for choosing the most efficient method.
Substitution — best when one variable is already isolated
1. Solve one equation for one variable
2. Substitute that expression into the other equation
3. Solve for the remaining variable
4. Back-substitute to find the first variable

Best for: y = ... or x = ... equations
Elimination — best when coefficients are easy to match
1. Multiply one or both equations so a variable has equal (or opposite) coefficients
2. Add or subtract the equations to eliminate that variable
3. Solve for the remaining variable
4. Substitute back to find the other variable

Best for: Ax + By = C form
Worked example — elimination
Solve: 3x + 2y = 16 and x − 2y = 0

Add equations: (3x + 2y) + (x − 2y) = 16 + 0
4x = 16 → x = 4
Substitute: 4 − 2y = 0 → y = 2
Solution: (4, 2)
Check in eq 1: 3(4) + 2(2) = 16 ✓   Check in eq 2: 4 − 2(2) = 0 ✓
Three possible outcomes — what they mean geometrically
One solution
Lines intersect at exactly one point. Different slopes.
y = 2x + 1
y = −x + 4
→ solve: (1, 3)
No solution
Lines are parallel. Same slope, different y-intercepts. Variables cancel leaving a false statement.
y = 2x + 1
y = 2x + 5
→ 0 = 4 (false)
Infinite solutions
Lines are identical. Same slope and same y-intercept. Variables cancel leaving a true statement.
y = 2x + 1
2y = 4x + 2
→ 0 = 0 (always true)
SAT strategy tips
Look for the easier variable to eliminate first
If the coefficients of one variable are already equal or opposite in the two equations, use elimination immediately — no multiplication needed. This saves 30–60 seconds per problem.
The SAT often asks for x + y, not x or y alone
Many SAT systems questions ask for a combination like x + y or 2x − y. After eliminating one variable, check if you can get the answer directly without finding both variables separately.
No solution: same slope, different intercept
For "no solution" questions, set the equations in slope-intercept form. If slopes are equal but y-intercepts differ, the system has no solution. The SAT frequently tests this condition.
Always check both equations
After solving, substitute your (x, y) values into both original equations. If both are satisfied, you're done. If not, you made an arithmetic error somewhere — find and fix it.
Practice questions
System of equations
2x + 3y = 12 4x − 3y = 6
The y-coefficients are already opposites (+3 and −3). Use elimination — add the equations directly to eliminate y, then solve for x.
Question 1 · Elimination Method
What is the solution (x, y) to this system?
A(3, 2)
B(2, 3)
C(1, 10/3)
D(4, 4/3)
Correct! (3, 2)
Add: (2x + 3y) + (4x − 3y) = 12 + 6
6x = 18 → x = 3
Sub into eq 1: 2(3) + 3y = 12
6 + 3y = 12 → y = 2
Check eq 2: 4(3) − 3(2) = 12 − 6 = 6 ✓
Not quite. Notice the y-terms are +3y and −3y — they're already set up for elimination! Add the two equations directly. The y's cancel, giving you 6x = 18. Solve for x, then substitute back to find y.
Add equations → 6x = 18 → x = 3
2(3) + 3y = 12 → y = 2 → (3, 2)
Problem
At a school event, adult tickets cost $8 each and student tickets cost $5 each. A total of 200 tickets were sold for $1,150 in revenue.
a + s = 200 8a + 5s = 1150
Let a = adult tickets, s = student tickets. Solve this system to find each quantity.
Question 2 · Real-World System
How many adult tickets were sold?
A50
B100
C150
D75
Correct! 50 adult tickets.
From eq 1: s = 200 − a
Sub into eq 2: 8a + 5(200 − a) = 1150
8a + 1000 − 5a = 1150
3a = 150 → a = 50
Check: s = 150 · 8(50)+5(150)=400+750=1150 ✓
Not quite. Use substitution: from equation 1, write s = 200 − a. Substitute into equation 2 and solve for a.
8a + 5(200 − a) = 1150
3a = 150 → a = 50
System with parameter
6x − 2y = 10 kx − y = 5
Find the value of k that makes this system have infinitely many solutions. Hint: for infinite solutions, the two equations must be identical — one must be a scalar multiple of the other.
Question 3 · Infinite Solutions Condition
For what value of k does this system have infinitely many solutions?
Ak = 6
Bk = 2
Ck = 3
Dk = 1
Correct! k = 3
For infinite solutions, eq 2 must be a multiple of eq 1.
Divide eq 1 by 2: 3x − y = 5
This matches eq 2 exactly when k = 3.
Check: 3x − y = 5 is identical to eq 1 ÷ 2 → infinite solutions ✓
Not quite. For infinite solutions, both equations must describe the same line — one must be a multiple of the other. Try dividing equation 1 by 2 and see what you get. That result should match equation 2 exactly.
Eq 1 ÷ 2: 3x − y = 5
This equals eq 2 when k = 3 ✓
