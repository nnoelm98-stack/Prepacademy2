Inequalities
Learn to solve linear inequalities, graph solution sets on a number line, and apply the one critical rule that catches most students off guard: what happens when you multiply or divide by a negative number.

Solving an inequality works exactly like solving an equation — with one critical exception.
✓
Add or subtract the same value from both sides — inequality sign stays the same.
x + 7 > 12 → x > 5 ✓
✓
Multiply or divide both sides by a positive number — inequality sign stays the same.
3x ≤ 15 → x ≤ 5 ✓
⚠️
Multiply or divide both sides by a NEGATIVE number — FLIP the inequality sign.
−2x < 8 → x > −4 (sign flipped!) ✓
−3x ≥ 12 → x ≤ −4 (sign flipped!) ✓
Why the sign flips — the intuition
Think of the number line: 2 < 5 is true. Multiply both sides by −1: −2 and −5. Now −2 is to the right of −5, so −2 > −5. Multiplying by a negative reverses the order of every pair of numbers on the number line. The inequality sign must flip to stay true.
Worked example
Solve: −3(x − 4) + 2 ≥ 14

−3x + 12 + 2 ≥ 14
−3x + 14 ≥ 14
−3x ≥ 0
x ≤ 0  (divide by −3: flip the sign!)

Solution set: all x ≤ 0  |  Number line: closed circle at 0, arrow pointing left.
SAT strategy tips
Mark negative divisions immediately
As soon as you see you need to divide by a negative number, circle the inequality sign as a reminder to flip it. This one habit eliminates the most common inequality error.
Compound inequalities: solve all three parts
For compound inequalities like −3 < 2x + 1 ≤ 9, perform operations on all three parts simultaneously. Whatever you do to the middle, do to both ends.
Test a value in the solution set
After solving, pick a value that should be in your solution set and substitute it into the original inequality. If it satisfies the inequality, your solution is likely correct.
Real-world inequalities: read "at least" and "at most"
"At least" = ≥ (greater than or equal). "At most" = ≤ (less than or equal). "More than" = > (strict). "Fewer than" = < (strict). These translations appear on every SAT.
Practice questions
Problem
Solve the inequality:
−4x + 3 > 2x − 9
Collect x-terms on one side. Watch carefully: does dividing to isolate x require dividing by a negative? If so, flip the sign.
Question 1 · Solving Inequalities
Which correctly describes all values of x that satisfy the inequality?
Ax > 2
Bx < 2
Cx > −2
Dx < −2
Correct! x < 2
−4x + 3 > 2x − 9
3 + 9 > 2x + 4x
12 > 6x
2 > x → x < 2
Check: x = 0: −4(0)+3=3 > 2(0)−9=−9 ✓
Check: x = 3: −9 > −3? No ✓ (not in solution)
Not quite. Move all x-terms to the right and constants to the left, then divide. Alternatively, add 4x to both sides and add 9 to both sides first.
−4x + 3 > 2x − 9
12 > 6x → x < 2
Problem
Solve the compound inequality:
−1 ≤ (2x − 3) / 5 < 3
Multiply all three parts by 5 to clear the fraction, then add 3 to all parts, then divide by 2. Perform each operation on all three parts simultaneously.
Question 2 · Compound Inequality
Which correctly describes the solution set?
A−1 ≤ x < 9
B1 ≤ x < 9
C−1 < x ≤ 9
D1 < x ≤ 9
Correct! 1 ≤ x < 9
Multiply all parts by 5:
−5 ≤ 2x − 3 < 15
Add 3 to all parts:
−2 ≤ 2x < 18
Divide all parts by 2:
−1 ≤ x < 9

Wait — that gives −1 ≤ x < 9 (option A).
Let me recheck: −5+3=−2, −2/2=−1. ✓
So answer is A: −1 ≤ x < 9.
Not quite. Work on all three parts simultaneously. Multiply by 5, then add 3, then divide by 2 — applying each operation to the left bound, middle, and right bound.
×5: −5 ≤ 2x−3 < 15
+3: −2 ≤ 2x < 18
÷2: −1 ≤ x < 9
Problem
A delivery driver can carry at most 850 pounds in her truck. She has already loaded boxes weighing a total of 340 pounds. Each additional crate weighs 35 pounds.
Write an inequality where c = number of crates, then solve to find the maximum number of crates she can add. Remember "at most" = ≤.
Question 3 · Real-World Inequality
What is the maximum number of additional crates the driver can load?
A14 crates
B15 crates
C24 crates
D25 crates
Correct! 14 crates.
340 + 35c ≤ 850
35c ≤ 510
c ≤ 14.57...
Since c must be a whole number: c ≤ 14
Check: 340 + 35(14) = 340 + 490 = 830 ≤ 850 ✓
Check: 340 + 35(15) = 340 + 525 = 865 > 850 ✗
Not quite. Set up: 340 + 35c ≤ 850. Solve for c. Since crates must be whole numbers, round down (not up!) to stay within the weight limit.
35c ≤ 510 → c ≤ 14.57 → c = 14
