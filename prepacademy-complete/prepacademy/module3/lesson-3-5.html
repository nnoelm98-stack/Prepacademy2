Linear Inequalities & Systems
Learn to graph linear inequalities in two variables, identify solution regions, and solve systems of inequalities — including the word problems where two constraints must both be satisfied simultaneously.
13 min read3 practice questions+20 pts on completion
Graphing a linear inequality in two variables
An inequality like y > 2x − 1 defines a region of the coordinate plane — all points (x, y) where the inequality holds. Graphing it involves two steps: draw the boundary line, then shade the correct side.
1
Draw the boundary line by replacing the inequality with an equals sign (y = 2x − 1). Use a dashed line for strict inequalities (< or >) and a solid line for ≤ or ≥.
Dashed = not included · Solid = included (on the boundary)
2
Pick a test point not on the line — (0, 0) works when the line doesn't pass through the origin. Substitute into the inequality.
y > 2x − 1 : test (0,0) → 0 > −1 ✓ → shade the side containing (0, 0)
3
Shade the correct half-plane. If the test point satisfies the inequality, shade the side containing it. If not, shade the opposite side.
y > mx + b (above the line)
Shade above the dashed boundary line. All points above have a larger y-value than the line at that x.
y > 2x − 1: shade above the dashed line y = 2x − 1
y < mx + b (below the line)
Shade below the dashed boundary line. All points below have a smaller y-value than the line at that x.
y < 2x − 1: shade below the dashed line y = 2x − 1
Systems of inequalities — the overlapping region
A system of inequalities requires finding all points that satisfy all inequalities simultaneously. The solution is the region where all shaded areas overlap. The SAT usually asks you to identify a point that is or isn't in this region.
Worked example — testing a point
System: y ≤ −x + 4 and y ≥ 2x − 2

Does (1, 2) satisfy the system?
Test ineq 1: 2 ≤ −(1) + 4 → 2 ≤ 3 ✓
Test ineq 2: 2 ≥ 2(1) − 2 → 2 ≥ 0 ✓
Yes — (1, 2) satisfies both inequalities and is in the solution region.

Does (3, 3) satisfy the system?
Test ineq 1: 3 ≤ −3 + 4 → 3 ≤ 1 ✗
No — (3, 3) fails the first inequality and is outside the solution region.
SAT strategy tips
The fastest SAT method: test the point directly
When asked which point lies in the solution region, substitute each answer choice into all the inequalities. The correct answer satisfies every inequality. No graphing required.
Dashed vs. solid — boundary inclusion
Strict inequalities (<, >) use dashed lines — boundary not included. Non-strict (≤, ≥) use solid lines — boundary included. The SAT tests this distinction in "which point is a solution" questions.
Word problems: write both constraints
SAT word problems with two conditions (budget limit AND minimum quantity, for example) require two inequalities. Write both before solving. The answer must satisfy both simultaneously.
Convert to slope-intercept to identify region
Before graphing, solve for y to get slope-intercept form. Then: y > → above the line; y < → below. This removes ambiguity about which side to shade.

Problem
A student is buying notebooks and pens for school. Notebooks cost $3 each and pens cost $1.50 each. She wants to spend at most $18 total and buy at least 4 notebooks.
Let n = notebooks, p = pens. Write two inequalities: one for the budget constraint, one for the notebook minimum.
Question 2 · System of Inequalities
Which system of inequalities correctly represents this situation?
A3n + 1.5p ≤ 18 and n ≥ 4
B3n + 1.5p ≥ 18 and n ≤ 4
C3n + 1.5p < 18 and n > 4
D3n + 1.5p = 18 and n = 4
Correct! "Spend at most $18" → total cost ≤ 18, so 3n + 1.5p ≤ 18. "At least 4 notebooks" → n ≥ 4. Both constraints use non-strict inequalities (≤ and ≥) because the boundary values are allowed — she can spend exactly $18 or buy exactly 4 notebooks. Option B reverses both signs. Option C uses strict inequalities excluding the boundary. Option D sets equalities instead of inequalities.
Not quite. Translate carefully: "at most $18" means the total can equal $18 but not exceed it — which inequality symbol captures that? "At least 4 notebooks" means 4 is the minimum allowed — which symbol captures that?
Problem
In the xy-plane, the point (k, 4) lies in the solution region of the following system:
2x − y > 3 x + 2y ≤ 14
Substitute (k, 4) into both inequalities and find the range of k-values that satisfy both simultaneously.
Question 3 · Finding Valid Parameter Values
Which value of k places (k, 4) in the solution region?
Ak = 2
Bk = 3
Ck = 4
Dk = 7
Correct! k = 4
Substitute y = 4 into both:
Ineq 1: 2k − 4 > 3 → 2k > 7 → k > 3.5
Ineq 2: k + 8 ≤ 14 → k ≤ 6
So: 3.5 < k ≤ 6

Check each option:
k=2: 2<3.5 ✗
k=3: 3<3.5 ✗
k=4: 3.5<4≤6 ✓
k=7: 7>6 ✗
Not quite. Substitute y = 4 into each inequality and solve for k. Inequality 1 gives a lower bound on k; inequality 2 gives an upper bound. Then test each answer choice to see which falls in the valid range.
Ineq 1: 2k−4 > 3 → k > 3.5
Ineq 2: k+8 ≤ 14 → k ≤ 6
